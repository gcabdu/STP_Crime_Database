About This Project

Purpose

This website was created to show crimes that have taken place in Saint Paul, Minnesota. It breaks down St. Paul into it's 16 neighborhoods, and breaking the crimes down into 12 different categories. The user can see the data on the map and also a table showing the data corresponding to that data point.

How

By using the filters on the side we are able to query for that specific data in our rest server that gives us back the appropriate data to show on the map and on the table below it. We also allow the user to alter the database that the data is pulled from by either inserting a new incident or deleting an old one.

Tools

This page was created usign HTML, Foundation CSS framework and custom CSS styling, and javascript with a Vue framework. This allowed us to make various components for the header, footer, map, and sidebar that allows use to easily create a stylied website.
