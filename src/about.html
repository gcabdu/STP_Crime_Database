About This Project

Purpose

This website was created to show crimes that have taken place in Saint Paul, Minnesota. It breaks down St. Paul into it's 17 neighborhoods, and breaking the crimes down into 12 different categories. The user can see the data on the map and also a table showing the data corresponding to that data point.

How to Use

By using the filters on the side we are able to query for that specific data in our rest server that gives us back the appropriate data to show on the map and on the table below it. We also allow the user to alter the database that the data is pulled from by either inserting a new incident or deleting an old one.

Tools

This page was created usign HTML, Foundation CSS framework and custom CSS styling, and javascript with a Vue framework. This allowed us to make various components for the header, footer, map, and sidebar that allows use to easily create a stylied website. We are able to pass the data back and forth between these components. The header goes into the App conponent where it shows the main header for the page, the footer is it's own stand along componenet that has a nice title in it, along with a link to this page. There also is a component that is for searching and filtering the data that the user wishes to see. We have created another componenet for inputting a URL to connect a server to the site in order to collect that data. Lastly we have map module, which has the legend component that shows the user what row color represents, and the last component are the rows themselves that we are able to iterate over the data in order to create as many rows as needed.
